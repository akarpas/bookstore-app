# BOOKSTORE APP

### DEPLOYED
[View](https://bookstore-app.netlify.com/)

### RUNNING LOCALLY

##### CLIENT

Node version is lts/carbon

```yarn install && yarn start```

Client will run on localhost:8080

### TESTING CLIENT
```yarn install && yarn test``` or ```yarn install && yarn test:watch```
